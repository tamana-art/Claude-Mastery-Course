## How to Use This Course

1. **Open a module** — Each module is in `modules/`
2. **Paste the activation prompt** at the top of each module to Claude Code
3. **Claude becomes your tutor** — It will ask YOU questions (Socratic method)
4. **Do the exercises** — They're hands-on, right in your terminal
5. **Take notes** — Your `notes/` folder is yours to fill

## Ground Rules

- There are no stupid questions. Ask anything.
- If Claude asks you a question, try to answer before asking for the answer.
- Struggle is learning. Don't skip the exercises.
- You can always say: "Explain that differently" or "Go deeper on that"
